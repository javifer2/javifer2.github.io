Microsoft Sql Server y afines.

¿Como eliminar los datos de una tabla y vaciar la base de datos de los mismos?

Aunque la pregunta parezca obvia, no lo es tanto y un proceso que a priori parece sencillo, puede convertirse en un handicap de desarrollo muy importante.

¿Que hacemos si nuestra base de datos, despues de intentar/realizar un script que incluye un delete sobre gran cantidad de filas, se vuelve más lenta?

Voy a ejecutar la demostración partiendo de una base de datos de prueba, que se ejecuta en un entorno Microsoft Sql Server 2016 Developer Edition, si bien todas las pantallas, y por supuesto, todo lo que os cuente en esta demo, se puede hacer en cualquier Sql server al menos versión 2008 o 2008R2, 2012, 2014, en cualquiera de sus ediciones, bien sea Express, Web, Standard, Developer y Enterprise. Muy probablemente no difiera el modo en 2005, pero estos últimos meses ya no veo ninguno, por tanto declino esta opción.
