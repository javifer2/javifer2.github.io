Próologo

La sintaxis básica de una sentencia SELECT, es tal que así:

SELECT * FROM ALUMNOS WHERE ID > 1

Donde lo primero siempre es el comando a ejecutar, despues sobre que columnas de la base de datos, luego la Clausula, para delimitar sobre quien o quienes son las tablas para obtener resultados. Posteriormente el criterio de exclusión

Según se avanza con las sentencias, cada vez se va a complicar más. Pero ya que estamos de faena:

Escribe una sentencia que busque los alumnos que tengan un Id mayor que 3, y que devuelva su Nombre y primer Apellido exclusivamente

Recuerda que las sentencias no se tienen porque escribir en una sola linea. Además no tienen porque ser en mayusculas, Y aunque no es obligatorio, se suele terminar con punto y coma

Inclusión de expresiones

Se pueden devolver expresiones formadas como campos de una tabla o mejor dicho de una consulta, incluyendo las mismas separadas por apóstrofes.

El planteamiento es que necesito esta lista en el siguiente formato.

  • Fernández - Rio, Marisa
  • Fernández - Jonhson, Ana
  • Fernández - Vázquez, Sedrick
  • Ferdesa - Bahamonde, Noa
  • Fernández - Cortés, Clara

select APELLIDO1 + '-'+ APELLIDO2 + ', '+ NOMBRE from ALUMNOS where APELLIDO1 like 'Fernández' OR ID=17

El concepto expresión en SQL, abarca muchas posibilidades.
